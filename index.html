STATION OPERATION GUIDE

  1. Data Input: Upload an image containing GPS metadata to auto-lock the location automatically and click on GENERATE FORENSIC PDF REPORT. All maps, satellite imagery, and analysis grids are automatically captured and included in your generated PDF report.
  2. Triangulation: If the system returns a Metadata Missing alert (NO GPS METADATA STRIPPED OR MISSING HEADERS), please proceed to the Manual Triangulation. Input the incident time and date manually to resolve the location via shadow analysis.
  3. Manual Analysis: Click 3 points on the image canvas:
    Point 1: Top of Object | Point 2: Base of Object | Point 3: End of Shadow.
  4. Triangulation: Enter the incident date and time, then click "Run Triangulation" to estimate coordinates.
  5. Export: Click "GENERATE FORENSIC PDF REPORT" to compile satellite evidence and analysis into a secure PDF. All maps, satellite imagery, and analysis grids are automatically captured and included in your generated PDF report.

ABOUT FORENSIC STATION

Forensic Station is a professional-grade web utility designed for digital forensic investigators and security researchers. This platform integrates metadata extraction, geospatial mapping, and geometric shadow analysis into a single browser-based environment. All maps, satellite imagery, and analysis grids are automatically captured and included in your secure PDF report.

Our mission is to provide accessible, high-fidelity investigative tools to the global security community.

PRIVACY POLICY

1. Data Processing: All forensic analysis is performed locally in your browser. We do not upload your evidence files to our servers.

2. Third-Party Services: We use OpenStreetMap and Esri for satellite data. These providers may collect IP addresses for tile delivery.

3. Local Storage: The application may use browser memory temporarily to process reports, which is cleared upon closing the session.
